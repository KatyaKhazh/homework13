# VirusTotal API: проверка файла по хэшу

Скрипт отправляет запрос в VirusTotal API по хэшу файла и выводит результат проверки.

## Что делает скрипт

Скрипт:

- берёт API-ключ из переменной окружения VT_API_KEY
- отправляет запрос в VirusTotal API
- получает информацию о файле по его хэшу
- выводит статус проверки, например:
  - файл malicious
  - файл suspicious
  - явных признаков вредоносности не найдено

---

## Требования

Перед запуском должны быть установлены:

- Python 3
- библиотека requests

Установка библиотеки:

bash
pip install requests
